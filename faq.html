全局标签调用

全局标签是的懒人DIY系统标签之一，在任何地方都可以调用，优先于其他标签系列。
{$webname} 作用 网站名称
{$webkey} 作用 网站关键词
{$webdesc} 作用 网站描述
{$seotitle} 作用 网站SEO标题
{$domain} 作用 网站域名
{$beian} 作用 网站备案号
{$tel} 作用 电话号码
{$qq} 作用 QQ
{$email} 作用 邮箱
{$adress} 作用 地址
{$copyright} 作用 版权信息
{$catename} 作用 分类名
{$catetitle} 作用 分类SEO标题
{$cateurl} 作用 分类URL
{$catekey} 作用 分类关键词
{$catedesc} 作用 分类描述
{$catelogo} 作用 分类logo
{$textcate} 作用 文章分类列表(使用说明请查看列表循环说明)
{$piccate} 作用 图片分类列表(使用说明请查看列表循环说明)
{$newpic} 作用 最新图片列表(使用说明请查看列表循环说明)
{$recpic} 作用 推荐图片列表(使用说明请查看列表循环说明)
{$toppic} 作用 置顶图片列表(使用说明请查看列表循环说明)
{$hotpic} 作用 热门评论图片列表按点击(使用说明请查看列表循环说明)
{$hotrevpic} 作用 热门评论图片列表(使用说明请查看列表循环说明)
{$newinfo} 作用 最新文章信息列表(使用说明请查看列表循环说明)
{$recinfo} 作用 推荐文章信息列表(使用说明请查看列表循环说明)
{$topinfo} 作用 置顶文章信息列表(使用说明请查看列表循环说明)
{$hotinfo} 作用 热门文章列表按点击(使用说明请查看列表循环说明)
{$hotrevinfo} 作用 热门评论文章列表(使用说明请查看列表循环说明)
{$pagelist} 作用 单页列表(使用说明请查看列表循环说明)
{$link} 作用 友情链接列表(使用说明请查看列表循环说明)

分类循环标签调用

下面是分类循环，不同的分类可以用上面的不同的标签比如：下面的是文章则是name="textcate" 图片则是name="piccate"
< volist name="textcate" id="lanren" >
{$lanren.id} 作用分类ID
{$lanren.cateurl} 作用分类URL
{$lanren.catename} 作用分类名称
{$lanren.catetitle} 作用分类SEO标题
{$lanren.catekey} 作用分类关键词
{$lanren.catedesc} 作用分类描述
{$lanren.catelogo} 作用分类logo
< /volist>

信息循环标签调用

信息包括文字和图片分类都是一样的用！在分类页可以用name="list",同时全局信息列表类标签也是可以用以下标签；比如要循环10条推荐的图片信息列表 name="recpic" offset="0" length="10"
< volist name="list" id="lanren">
{$lanren.id} 作用文章ID
{$lanren.title} 作用信息标题
{$lanren.url} 作用信息URL
{$lanren.istop} 作用置顶信息（1为置顶0为不置顶）
{$lanren.isrec} 作用推荐信息（1为推荐0为不推荐）
{$lanren.isrev} 作用是否允许评论(1为允许0为不允许)
{$lanren.infotitle} 作用信息SEO标题
{$lanren.infokey} 作用信息关键词
{$lanren.infodesc} 作用分类描述
{$lanren.content} 作用信息内容
{$lanren.ispic} 作用信息缩略图
{$lanren.author} 作用信息作者
{$lanren.from} 作用信息来源
{$lanren.revs} 作用信息评论数量
{$lanren.hits} 作用信息点击量
{$lanren.time} 作用信息发布时间
{$lanren.phoneurl} 作用预留字段
{$lanren.pcurl} 作用预留字段
< /volist>

评论列表标签调用

用于信息内容页，使用举例
< if condition="$isrev eq 0">(如果后台没开启评论显示)不允许评论

< else/> < volist name="commentlist" id="revs">
{$revs.homepage} 作用主页
{$revs.name} 作用评论者称呼
{$revs.time} 作用评论时间
{$revs.connent} 作用评论内容
< if condition="$revs.reply ">作用(如果有回复)
{$revs.replytime}作用回复时间
{$revs.reply}作用回复内容
< /if> < /volist>
发表评论POST地址：__GROUP__/Index/comment
称呼 name="name"
E-mail name="email"
主页 name="homepage"
内容 name="connent"
验证码 src="__GROUP__/code"
< /if>
